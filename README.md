# Senegal Fashion Market Dashboard 2025  

Live app → https://senegalfashionmarketapp2027.streamlit.app/

![App preview](https://raw.githubusercontent.com/nguemtchuengdanielle/Senegal_Fashion_Market_App/master/background.jpg)

## Description du projet

Cette application web permet de **scraper en temps réel** les annonces de vêtements et chaussures (hommes & enfants) sur **CoinAfrique Sénégal**, puis d’afficher et analyser les données de façon interactive.

Fonctionnalités principales :
- Scraping en direct (BeautifulSoup + requests)
- Tableau de données avec téléchargement CSV
- Dashboard interactif avec 4 graphiques Plotly
- Design professionnel et responsive (mobile-friendly)
- Déploiement 100 % gratuit sur Streamlit Cloud

## Lien de l’application

https://senegalfashionmarketapp2027.streamlit.app/

## Aperçu des pages

1. **Scrape data using BeautifulSoup** → 4 boutons pour récupérer les annonces en live  
2. **Dashboard of the data** → 4 graphiques interactifs (histogramme, boxplot, top 10 villes, villes les plus chères)  
3. **Download scraped data** → téléchargement des anciens fichiers  
4. **Evaluate the App** → formulaires de feedback (Google Forms + KoboToolbox)

## Comment tester l’application

1. Ouvrez le lien : https://senegalfashionmarketapp2027.streamlit.app/
2. Choisissez le nombre de pages (1 à 20)
3. Cliquez sur un bouton (ex: Men’s Clothes)
4. Attendez quelques secondes → les données apparaissent
5. Passez au Dashboard → explorez les graphiques
6. Téléchargez le CSV si vous voulez
